# Release History

This document provides a complete record of all releases of Please Don't Destroy. The document is ordered temporally, most recent release first. This application observes semantic versioning. This means:

- Major releases (e.g. 1.0.0 => 2.0.0) bring new features but may break the app. Make sure to review these notes.
- Minor releases (e.g. 1.0.0 => 1.1.0) introduce enhancements without breaking existing functionality.
- Patch releases (e.g. 1.0.0 => 1.0.1) fix bugs only.

## 0.3.x

Published 4th December 2025.

- Development version(s).
- Creates a "Mod Notification" when a new version is available.
  - "Major Update", "New Feature", and "Bug Fix" options.
  - "Critical" patches are **always** notified.
- Identifies banned/suspended users and removes associated data from the store.
